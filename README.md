# UAV Formation Flying - Passive Positioning System

## 整体建模思路 (Overall Modeling Approach)

本项目解决无人机编队飞行中的纯方位无源定位问题，通过数学建模和Python程序实现无人机位置调整方案。

### 核心问题分析
1. **纯方位无源定位**: 利用方向信息进行无人机定位
2. **编队队形保持**: 圆形编队和锥形编队的位置调整
3. **信号发射策略**: 最小化发射信号的无人机数量

### 数学建模方法
- **几何建模**: 基于角度测量的三角定位
- **优化算法**: 位置调整的迭代优化
- **信号处理**: 方向信息的提取和处理
